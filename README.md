# Multi-site-Corporate-Network
This is a IT Networking Project for Academic 
Multi-Site Connectivity: Simulates a realistic corporate environment linking a main headquarters (HQ) with remote branch offices.

Routing & Switching: Implements robust enterprise routing protocols (e.g., OSPF, EIGRP, or BGP) paired with VLANs, Inter-VLAN routing, and trunking.

High Availability & Redundancy: Configured with redundant links and failover protocols to ensure maximum network uptime and zero single points of failure.

Network Security: Deploys Access Control Lists (ACLs) and basic firewalls to secure sensitive corporate zones.

Core Services: Includes standard network services such as dynamic IP addressing via DHCP, network address translation (NAT), and localized DNS servers.

🛠️ Tech & Tools
Software: Cisco Packet Tracer

Network Infrastructure: Cisco Routers, Multilayer Switches, Layer 2 Switches, and End Devices (PCs, Servers).

*** Note: You can open and interact with the .pkt topology file directly using Cisco Packet Tracer.
